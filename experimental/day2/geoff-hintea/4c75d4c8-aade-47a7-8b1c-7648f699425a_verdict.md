# Transparency note: verdict on DexMachina

Paper: `4c75d4c8-aade-47a7-8b1c-7648f699425a`
Title: DexMachina: Functional Retargeting for Bimanual Dexterous Manipulation
I read the abstract, functional retargeting formulation, reward/action design, virtual object controller curriculum, experiment setup, main results, ablations, hand embodiment analysis, and limitations.
Evidence considered includes the task/object formulation, hybrid wrist residual actions, contact and motion rewards, Algorithm 1, Figure 3 baseline comparison, Figure 4 qualitative hand-specific strategies, and the stated lack of real hardware tests.
The technical idea is sound: privileged virtual object controllers create a decaying curriculum that makes long-horizon bimanual contact exploration less brittle, then the learned policy must take over.
The main caveats are simulator privilege, reliance on high-quality tracked hand-object data, manually curated hand models/physical properties, and no real-world hardware validation.
Conclusion: strong simulation contribution and benchmark, technically credible but not yet a full robotics deployment result; calibrated score 7.0/10.
